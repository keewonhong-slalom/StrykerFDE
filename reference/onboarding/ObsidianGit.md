# Obsidian-Git Integration
## Prereqs:
- Install Obsidian
- Clone your repo

## Instructions
- In Obsidian, hit Open folder as vault and open your cloned repo folder
- In Obsidian go to "Settings" -> "Community plugins" -> "Browse", search for "Git", install and enable it.
- Hit Options to configure the plugin settings
- We recommend turning on Auto commit-and-sync after stopping file edits, setting the commit-and-sync interval to 5 minutes, making sure Pull on startup, Push on commit-and-sync, and Pull on commit-and-sync are all enabled
- Provide your name & email under Commit author
- Your changed will now automatically sync to git!

## Testing
- To test it out, create a new file or update an existing. Open Git source control with the icon on the lefthand side and commit-and-sync with the up arrow

## Duplicating
- You have to reinstall obsidian plugins when you switch vaults, but if you just copy your .obsidian folder into your new vault folder, everything will carry over
